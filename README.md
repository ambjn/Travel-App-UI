# Travel App UI

## About:

Travel App UI is a modern & elegant UI made using Flutter.

## Packages:

<ul>
<li> font_awesome_flutter: ^10.1.0 </li>
</ul>

### Installation:

In the command terminal, run the following commands:

    $ git clone https://github.com/ambjn/TravelApp-UI.git
    $ cd TravelApp-UI
    $ flutter run
